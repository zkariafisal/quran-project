# 🕌 Quran Project - المصحف الكريم

مشروع واجهة تفاعلية لقراءة وتلاوة القرآن الكريم، مبني على React.

## 🌐 المشروع المباشر

[https://quran-project.vercel.app](https://quran-project.vercel.app)

## 📦 التقنيات المستخدمة

| التقنية | الإصدار | الاستخدام |
|---------|---------|-----------|
| React | 15.4.1 | واجهة المستخدم |
| Redux | 3.5.2 | إدارة الحالة |
| Webpack | 2.7.0 | بناء المشروع |
| Express | 4.14.0 | الخادم الخلفي |
| Sass | 1.77.0 | تنسيق CSS |

## 🚀 التشغيل المحلي

### المتطلبات الأساسية
- Node.js 18.x أو أحدث
- npm 9.x أو أحدث

### الخطوات

```bash
# 1. استنساخ المشروع
git clone https://github.com/zkariafisal/quran-project.git
cd quran-project

# 2. تثبيت الاعتمادات
npm install --legacy-peer-deps

# 3. تشغيل وضع التطوير
npm run dev

# 4. فتح المتصفح على الرابط
http://localhost:8000
