# 🕌 Quran Project

مشروع القرآن الكريم - واجهة تفاعلية لقراءة القرآن

## 🌐 المشروع

هذا المشروع هو نسخة مطورة من [quran.com-frontend](https://github.com/quran/quran.com-frontend) مع تعديلات للنشر على Vercel.

**المستودع الخاص بي:** [github.com/zkariafisal/quran-project](https://github.com/zkariafisal/quran-project)

## 🚀 التشغيل المحلي

```bash
# تثبيت الاعتمادات
npm install --legacy-peer-deps

# تشغيل وضع التطوير
npm run dev

# بناء المشروع للإنتاج
npm run build
